如何使用

1. 安装gnome-shell-extension manager
   
    <code>https://mattjakeman.com/apps/extension-manager</code>
2. 下载源码，并解压，拷贝到 `~/.local/share/gnome-shell/extensions/`，并重命名为 `amd-gpu-rocm-info@black-hat`
   
   ![screen](./assets/guide.jpg)

3. 打开 Extension Manager

    ![screen](./assets/guide2.jpg)
   
5. 登出，重新登录
   
6. 效果
    
    ![screen](./assets/screen.jpg)
